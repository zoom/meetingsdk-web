# zoomus-jssdk
zoom webclient jssdk Local NPM repository

[![NPM](https://nodei.co/npm/zoomus-jssdk.png)](https://nodei.co/npm/zoomus-jssdk/)

```
npm i zoomus-jssdk
```

[sample-app-web](https://github.com/zoom/sample-app-web) is a demo how to use zoomus-jssdk.
